git init - inicializar um repositório remoto

git clone - faz o download de um repositório

git status - apresenta o status do repositório

git add <nome do arquivo> - adiciona um arquivo para o próximo commit

git add . - adiciona todos os arquivos (untracked e modified) para o próximo commit

git commit -m'comentário' - captura um instantâneo das mudanças preparadas do projeto no momento

git push -  enviar os commits de sua branch e repositório Git local para o seu repositório remoto.
